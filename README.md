# House Price Data Analysis (Python)

## Overview
This project performs exploratory data analysis (EDA) on residential house pricing data using Python.  
The goal is to understand the key factors influencing house prices, identify hidden patterns in the housing market, and generate data-driven insights through statistical analysis and visualization.

---

## Dataset
The dataset (`house_prices.csv`) includes structural, spatial, and quality-related features of residential properties, such as:

- **SalePrice** → Final sale price of the property  
- **LotArea** → Lot size in square feet  
- **OverallQual** → Overall material and finish quality  
- **YearBuilt** → Original construction year  
- Additional structural, neighborhood, and condition-based attributes
